# Employee Attrition Analysis

## Project Overview
This project analyzes employee attrition data to identify patterns, trends, and factors affecting employee retention within an organization.

The project was developed using SQL and Python to perform data cleaning, analysis, and business insight generation from HR employee data.

---

# Objectives
- Analyze employee attrition behavior
- Identify departments with high attrition
- Study salary and experience trends
- Analyze job satisfaction levels
- Understand workforce distribution
- Generate HR-related business insights

---

# Tools & Technologies Used
- Python
- SQL
- pandas
- matplotlib
- MySQL

# Key Insights
- Sales department showed higher attrition compared to other departments.
- Employees with lower job satisfaction were more likely to leave.
- Employees with lower salaries had higher attrition rates.
- Long-term employees showed better retention.
- Younger employees changed jobs more frequently.
